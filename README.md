# 🎓 Grade12GuideSystem

A modern web-based platform designed to help Grade 12 students prepare for entrance exams, explore university opportunities, and track their academic progress efficiently.

Built to support students with smart learning tools, practice resources, and career guidance in one place.

---

## 🚀 Features

### 📚 Learning Resources
- Access digital books and study materials
- Organized educational content for Grade 12 students

### 📝 Entrance Questions & Answers
- Previous entrance exam questions
- Practice with solutions and explanations

### 🎯 University Requirement Checker
- Check required scores / eligibility for universities
- Compare available study options

### ✅ Practice Tests
- Interactive mock exams
- Self-evaluation system
- Instant results

### 📈 Progress Tracking
- Monitor completed tasks
- View learning progress
- Performance analytics

### 📅 Study Planner
- Smart daily / weekly study planning
- Time management tools for students

### 🏫 University Information
- Explore universities
- Programs and opportunities

## ⚙️ Installation Guide
# 1️⃣ Clone Repository
- git clone https://github.com/abrham-27/entrance_guide.git
# 2️⃣ Move Project Folder
- Place the project inside your local server directory:

  For XAMPP:

  htdocs/
# 3️⃣ Setup Database
- Open phpMyAdmin
 Create a new database:
 grade12guide
 - Import the provided SQL file
 
# 4️⃣ Configure Backend
 Update database connection settings in PHP config file:

host = localhost
user = root
password = ""
database = grade12guide

# 5️⃣ Run Frontend
cd frontend
npm install
npm start

# 6️⃣ Start Apache & MySQL

Run XAMPP / WAMP and open:

http://localhost/yourproject
